# Lab 1 — AWS S3 Static Website

## Overview
This lab demonstrates how to host a static website using Amazon S3.

## AWS Services Used
- Amazon S3
- Static Website Hosting
- Bucket Policy

## Steps Performed
1. Created an S3 bucket named `danielb-cloud-lab1-website`
2. Uploaded an `index.html` file
3. Enabled static website hosting
4. Configured a bucket policy to allow public read access
5. Accessed the website through the S3 website endpoint

## Result
A simple static webpage was successfully hosted using Amazon S3.

## Skills Demonstrated
- Cloud storage configuration
- Static website hosting
- S3 permissions and policies
